# 설정 오버레이 (Settings Overlay)

> 언어: [English](./en.md) · **한국어**
>
> 관련: [PR #137](https://github.com/yhk1038/claude-code-gui-jetbrains/pull/137)

## 새로운 기능

이제 **설정(Settings)을 열어도 진행 중인 세션이 끊기지 않습니다.** 예전에는 톱니바퀴를 눌러 설정을 열면 채팅 화면이 통째로 교체되어, **응답을 스트리밍 중이던 세션이 중단되고** 채팅 상태가 초기화됐습니다. 이제 설정은 현재 채팅 **위에 오버레이로 떠서**, 아래에서 세션이 그대로 살아 있습니다.

## 기능 상세

### 오버레이로 열리는 설정

- 상단바의 **설정(톱니바퀴) 버튼**을 누르면 설정이 현재 채팅 위에 오버레이로 표시됩니다.
- 오버레이가 떠 있는 동안에도 **아래의 채팅은 계속 마운트된 상태**입니다. 스트리밍 중이던 응답은 멈추지 않고 이어집니다.
- 오버레이는 다음 세 가지 방법으로 닫을 수 있습니다.
  - 우측 상단의 **X 버튼**
  - **Esc 키**
  - 오버레이 **바깥 영역(배경) 클릭**
- 닫으면 설정의 어느 하위 화면(Account, Usage 등)에 있었든 **한 번에 원래 세션으로 복귀**합니다.

### 여는 방식 선택 (오버레이 / 새 탭)

오버레이가 기본 동작이지만, 기존처럼 **설정을 전용 탭으로 여는 방식**을 선호한다면 바꿀 수 있습니다.

- **Settings → General → "Open settings as"** 항목에서 **Overlay** 와 **New tab** 중 선택합니다. (기본값: Overlay)
- **New tab** 을 고르면 설정이 별도의 에디터 탭에서 전체 화면으로 열립니다.

## 알아두기

- 설정을 전용 탭으로 열었다가 **뒤로 가기**를 눌러도, 빈 세션이 아니라 **직전에 보고 있던 세션**으로 정확히 돌아갑니다.
- 오버레이 방식에서는 설정을 여닫아도 현재 세션 ID가 바뀌지 않으므로, 세션 전환 시 일어나는 화면 초기화가 발생하지 않습니다.
